使用方式
========
1. 將 uboot.sh 放入內建 USB 第一個分割區, 並依情況編寫(寫錯無法開機)
2. usb 碟第一個分割區設定為 FAT/FAT32
3. 將 iptvubootupdate.bin 及 uboot.sh 放入 USB
4. 拔網樂通電源
5. 按住 reset 鍵不放
6. 連上網樂通電源
7. 等燈閃3下後
8. 放開 reset 鍵

恢復原版uboot方式
=================
1. 將 iptvubootupdate.org copy 到 USB, 並更名為 iptvubootupdate.bin
2. 拔網樂通電源
3. 按住 reset 鍵不放
4. 連上網樂通電源
5. 放開 reset 鍵
